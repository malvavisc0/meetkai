# kAI Persona

The default kAI WhatsApp persona. Casual friend vibe with full media support, organic group participation, and TTS voice notes.

## Transport

Waha

## Actions

reply, send_voice_note, silent, sleep, send_dm, send_to_group, console

## Tools

**Optional:** brain_query, sql_query, send_email, calcom, record_note, get_conversation_messages, get_whatsapp_history, schedule_task, list_tasks, cancel_task

## Required environment

- **brain_query**: KAI_BRAIN_BASE_URL, KAI_BRAIN_LIGHTRAG_API_KEY
- **sql_query**: KAI_SQL_DSN
- **send_email**: KAI_SMTP_TOOL_HOST, KAI_SMTP_TOOL_USERNAME, KAI_SMTP_TOOL_PASSWORD, KAI_SMTP_TOOL_FROM_ADDRESS
- **calcom**: KAI_CALCOM_API_KEY

## Quick start

Selectable from the deployment wizard; tool toggles on the settings page.
