# kAI Brainstorm

Creative collaborator on WhatsApp. Builds on your ideas, asks clarifying questions, and records session notes so follow-ups pick up where you left off.


## Transport

Waha

## Actions

reply, silent, console

## Tools

**Optional:** record_note, get_conversation_messages, brain_query

## Required environment

- **brain_query**: KAI_BRAIN_BASE_URL, KAI_BRAIN_LIGHTRAG_API_KEY

## Quick start



Selectable from the deployment wizard; tool toggles on the settings page.
